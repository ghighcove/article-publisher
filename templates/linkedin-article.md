## Template Guidelines

**Tone**: Professional but conversational. Write as if explaining to a smart colleague.

**Length**: 1,500-2,500 words for substantial articles; 800-1,200 for focused pieces.

**Formatting**:
- Short paragraphs (3-4 sentences max)
- Frequent subheadings for scannability
- Bold key terms on first use
- Use bullet points for lists
- Include code blocks for technical content

**Structure**:
- Hook in the first paragraph
- Promise value clearly
- Deliver on the promise with specific, actionable content
- End with clear next steps

**Links**:
- Link to primary sources and documentation
- Include 3-5 strategic links to relevant resources
- Link to your LinkedIn for connection

**SEO/Discoverability**:
- Use keywords naturally in headings
- Front-load important terms in first paragraph
- Include specific technical terms that people search for
